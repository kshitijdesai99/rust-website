How to Run the Backend (Axum):
From website1/: cargo run -p backend
Serves at http://127.0.0.1:3000 with endpoint GET /api/hello

Run the Frontend
Location: go into the frontend/ folder: website1/frontend/
One-time setup:
rustup target add wasm32-unknown-unknown
cargo install trunk
Start the dev server (in website1/frontend/):
trunk serve --open
Backend stays running at http://127.0.0.1:3000. The frontend will be at http://127.0.0.1:8080.
